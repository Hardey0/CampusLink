# CampusLink Client

React + Vite frontend for the CampusLink platform.

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` — Start dev server
- `npm run build` — Build for production
- `npm run preview` — Preview production build
- `npm run lint` — Run ESLint

## Stack

- React 19
- Vite
- Tailwind CSS
- React Router DOM
